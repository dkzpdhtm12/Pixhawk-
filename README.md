# Pixhawk
----------------------------------
자율 주행 선박 개발 중에 Pixhawk란 것을 알게 되었다. 젠장 자율 주행 관련 UI를 제공해 주다니 너무 어렸다.

그냥 지도에 좌표만 찍고 주행 버튼을 누르면 자동으로 이동한다. 쉽다 미쳤다. 또 ROS와 연동이 된다니 더욱 안 건드려 볼 수가 없었다.

* https://kwangpil.tistory.com/21 사이트를 통해서 기본적인 이론을 공부할 수 있었다.
